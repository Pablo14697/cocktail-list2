### Getting started

Follow the next steps:<br/>
- `git clone https://github.com/Pablo14697/cocktail-list2/edit/master/README.md`
- `cd cocktail-list2`
- `yarn`
- `react-native link`

That's it! ðŸŽ‰ 

init Android Emulator
- `react-native run-android`

init iOS Emulator
- `react-native run-ios`
